# Ayesha Mohapatra | Gamified Portfolio

Welcome to my interactive gamified portfolio! This project is a React-based web application that showcases my journey as a Software Development Engineer through an immersive, game-like experience.

## üåê Deployment

This project is ready to be deployed to any static hosting provider.

### Recommend Providers:

- **Vercel**: Simply connect your GitHub repository; it will automatically detect Vite and deploy.
- **Netlify**: Drag and drop the `dist` folder or connect via GitHub.
- **GitHub Pages**:
  1. Install `gh-pages`: `npm install gh-pages --save-dev`
  2. Add `homepage` to `package.json`.
  3. Add deploy scripts: `"predeploy": "npm run build", "deploy": "gh-pages -d dist"`
  4. Run `npm run deploy`.
